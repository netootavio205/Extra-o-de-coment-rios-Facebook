# Icons Directory

This directory contains the extension icons in different sizes:

- icon16.png - 16x16 pixels (for toolbar in small mode)
- icon32.png - 32x32 pixels (for toolbar and popup)
- icon48.png - 48x48 pixels (for extensions management page)
- icon128.png - 128x128 pixels (for Chrome Web Store)

All icons are based on the original ico.png file from the project root.
